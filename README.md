# Customer Churn Prediction

## Project Overview
This project predicts customer churn using machine learning techniques on the IBM Telco Customer Churn dataset.

## Objectives
- Analyze customer behavior
- Identify factors influencing churn
- Build predictive models
- Evaluate model performance

## Dataset
IBM Telco Customer Churn Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation

## Models Used
- Logistic Regression
- Random Forest
